So many great ways to start collecting Nanrs and monetize your content!

Not sure how to get started with the Nanr platform? Send us an email at support@nanr.io or head over to our contact page

I have a blog or website

Add this script tag to the head or body of your blog or webiste.
{{scriptTag}}
Then add this tag wherever you want a Nanr button to appear.
{{generateButtonTag(tag.id)}}
Optional Properties
nanr-sizeThe size of the button. Must be larger then 40 and defaults to 54
nanr-pageIdhis can be any string that identifies the page.

I have social media or other platform

Just direct people to this link and they will be taken to your Nanr stand where they can send you Nanrs
{{generateStandLink(tag.username)}}

I Have Premium Content

You can use the Nanr platform to block content until the user has reached a minimum donation.
Add this script tag to the head or body of your blog or webiste.
{{scriptTag}}
Call the require function. If the user hasn't reached the minimum donation a overlay will be rendered blocking your content and allowing the user to donate
{{generateRequireTag(tag.username)}}
username - This should be set to your username
pageCnt - This is minimum donation required to view the page
pageId - This can be any string that identifies the page.
title (optional) - Defines the title to display on the Nanr Wall
text (optional) - Defines the text to display on the Nanr Wall
background (optional) - Can be set to 'nanr', 'lines' or 'solid'. Defines the background pattern.

I Need a Custom Solution

We make it easy to integrate Nanr into any platform.
Follow the directions in the "I have a blog or website" section to add one or more Nanr buttons to your platform
Add this script to your page and the nanrEvent function will be called after each Nanr button is initialized or clicked
{{generateCallbackTag()}}
loggedIn - Will be set to true or false to specify if the user is logged into a Nanr account
pageNanrs - The number of Nanrs the current users has given on the specified page. Will be null if no user is logged in.
totalNanrs - The total number of Nanrs the current users has given to you. Will be null if no user is logged in.
pageId - The pageId specified in the nanr button
